# Pack 2: ビジネス自動化プロンプト集

**15+ Prompts for Business Operations and Automation**

$12 - Ready-to-use templates for email, meetings, analysis, and strategy

---

## 使い方

各プロンプトはビジネスシーンで実際に使える実用的なテンプレートです。

**活用方法**:
1. プロンプトテンプレートをコピー
2. `[...]` 部分を自分の状況に置き換え
3. Claude に貼り付けて実行
4. 出力結果を微調整して使用

## Prompt #1: メール自動下書き作成

**タイトル**: Email Drafting Assistant

**ユースケース**: メールを書く時間がない。要点だけ伝えて、プロフェッショナルなメールを自動生成したい。

**プロンプトテンプレート**:
```
以下の情報からプロフェッショナルなビジネスメールを作成してください。

【メール情報】
宛先: [相手の名前・役職]
主目的: [何をしたいのか: 提案/報告/質問/依頼など]
要点:
- [ポイント1]
- [ポイント2]
- [ポイント3]

【トーン】
[フォーマル/カジュアル/親切/厳格]

【関係性】
[初めて/既に何度も連絡済み/部下/上司/取引先]

【アクション】
相手に取ってほしい行動: [返信をもらう/承認をもらう/日程調整など]

期限: [あれば指定]

以下の形式で作成してください:
1. 件名
2. 本文 (挨拶〜本題〜クロージング)
3. 署名
4. 送信前チェックリスト
```

**出力例**:
```
【件名】Q2プロダクトロードマップについてのご相談

【本文】

鈴木様

いつもお世話になっております。
田中です。

Q2のプロダクト開発方針について、ご相談させていただきたくご連絡いたしました。

【提案内容】
現在、以下の3つの機能について優先度の判断が必要な状況です:
・ API レート制限機能 (開発期間: 2週間)
・ ダッシュボード UI リデザイン (開発期間: 3週間)

リソースの制約から、これら全てを並行実施できないため、貴社のご指示をいただきたいです。

【次のステップ】
恐れ入りますが、来週金曜日までにご優先度のご指示をいただけますでしょうか?
ご質問やご不明な点がございましたら、お気軽にお問い合わせください。

よろしくお願いいたします。

田中太郎
OpenClaw Inc.
```
